<p align="left">This project was created after the initial GameHub project was deleted as the GitHub account owning this project was taken down.<br><br>GameHub is a social gaming platform  delveloped using the MERN stack with a microservices architecture, enabling users to play games, chat, compete on leaderboards, and manage friends in a clean and efficient system.<br><br>This project contains Frontend and Backend of the application. Backend contains multiple microservices such as user-service, chat-service, games-service, etc.<br><br>Optimized performance by integrating Redis, reducing response time by 70%.<br>Enhanced security by implementing rate limiting, password encryption, input validation, and dynamic error handling, reducing security threats by 60%.<br><br>Minimized server load by 90% through the implementation of debouncing techniques.<br><br>Built core features including real-time messaging and gaming, player search based on location (Google Maps), password reset via Nodemailer, friend management, and more.<br><br>Leveraged technologies such as Socket.io, Redux Toolkit, Cloudinary, Google OAuth 2.0, and Nodemailer to enhance functionality and user experience.<br><br>Deployed the application on AWS EC2 using a custom domain, configured nginx for proxy routing, and secured the platform with HTTPS via SSL certificates.</p>
